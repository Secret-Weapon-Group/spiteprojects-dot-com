Privacy Policy

Last updated: February 2025

The Short Version

We collect minimal data. No tracking. No ads. No selling data.

What We Collect

When You Browse

  • Standard server logs (IP, user agent, pages visited)
  • Logs are deleted after 30 days
  • No cookies for tracking
  • No analytics scripts

When You Submit a Project

  • The git repository URL you provide
  • Metadata from your .spite or SPITE.md file
  • Your IP address (for abuse prevention, deleted after 30 days)

What We Don't Do

  • No third-party analytics (no Google Analytics)
  • No advertising
  • No selling or sharing data with third parties
  • No user accounts or passwords
  • No tracking across sites

External Services

We fetch metadata from:

  • GitHub API (to read .spite files and repo activity)
  • GitLab API (same)

These services have their own privacy policies.

Data Location

Data is stored on Google Cloud (US region).

Your Rights

Want your project removed? Email spiteprojects@secretweapon.group and we'll remove it within 48 hours.

Changes

We'll update this page if our practices change.

Contact

Questions? Email: spiteprojects@secretweapon.group
