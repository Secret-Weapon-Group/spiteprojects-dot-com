## Usage in Project Metadata

Add to your `.spite` file or `SPITE.md` frontmatter:

```yaml
badges:
  - speed-run
  - rage-quit
  - subscription-killer
```

## Badge Validation

- Must use valid badge IDs from the list above
- Maximum 5 badges per project (quality over quantity)
- Self-reported honor system (we trust your spite is genuine)
- Invalid badge IDs will fail validation
